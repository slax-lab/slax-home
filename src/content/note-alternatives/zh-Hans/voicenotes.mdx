## Voicenotes 做得好的地方

直说。Voicenotes 产品(现在实际是同一品牌下的三个产品:Meetings、Dictation、Memos)在范畴上让人印象深刻。

突出之处:

- **跨完整历史 Ask AI**。Pro 的「你的对话永久可搜」是真的第二大脑承诺
- **广平台覆盖**。macOS、Windows、iOS、Android、watchOS、Wear OS、Chrome 扩展、Web。这个品类里比多数对手都多
- **混合语言自动检测**——说话中切换,不用设置
- **转录范畴**——官网 60+ 语言,商店列表更多
- **「Create / AI Creations」**——8 个内置预设(摘要、待办清单、邮件、消息、会议报告等),加一个自定义 prompt 输入。保存的 Prompt 有 Custom 和 Saved 标签
- **Pro 集成**——Zapier、Obsidian、Notion、Claude、ChatGPT 开箱即用
- **会议录音 + 即时报告**——如果你真做会议笔记,这是头条功能
- **听写清理**——自动去掉填充词和重新开头

如果你的工作涉及会议,而你想要一个工具在一份订阅里处理会议 + 个人笔记 + 听写,Voicenotes 是强选择。

## Slax Note 走的不同

Voicenotes 拓宽到覆盖会议 + 听写 + memos + 集成。Slax Note 守在个人语音笔记上——并押了一个 Voicenotes 没押的具体注。

### 可以发给别人的 Prompt

两个产品都允许你写自定义 Prompt。Voicenotes 的 Saved Prompt 活在你自己账号里——你可以编辑、改名、删除,但它们不离开你账号。

Slax Note 的 Prompt 是**可分享的**。你写一条——*「把这个变成我的口吻的公众号开篇」*——可以发给朋友、队友或另一位公众号作者。他们也能用。

如果你在一群人里写作,而大家总在说*「能不能把你的 prompt 发我」*,我们做的就是这个工作流。

### 我们不做会议

Voicenotes 的首页现在主打 Meetings。Memos 是三个子产品之一。

Slax Note 不做会议。我们不做说话人分离、日历集成、参会者识别。如果会议是你的真活儿,Voicenotes(或 Otter、Granola、Fireflies、tl;dv)是更好的工具——我们宁愿告诉你这一点,也不假装我们在竞争。

我们**做的**是个人语音笔记——散步时的想法、日记式的反思、想要发布的初稿——由你(或你的朋友)写的 Prompt 来塑形。

### 价格

Voicenotes 的个人版(以 iOS 内购为准)是**14.99 美元/月**或**89.99 美元/年**(99.99 标准,89.99 促销)。还有**团队版每人 9 美元/月**(以官方帮助中心为准)。终身「Believer」档已下架。

Slax Note Pro 是**4.99 美元/月**或**49.99 美元/年**。

不同的产品,所以单纯比价格框架不对。但如果你不需要会议录音或 Pro 集成,你在为很多用不到的能力付钱。

> **「Voicenotes 是包含会议的三产品套件,Slax Note 是聚焦的个人语音笔记应用,Prompt 可分享。」**

## 我们怎么想这件事

两个信念塑造了 Slax Note。两个都和 Voicenotes 在 2026 年走的方向形成直接对比。

**个人优先,不做会议**。会议工具会把产品拉向说话人分离、日历集成、参会者识别、企业定价。每一条都把焦点从那个一边走在街上一边有想法的人身边拉远。Voicenotes 把这些都加上,做成大门。我们刻意没有。Slax Note 是给那个人的,不是给会议室的。

**做成五年后还在,陪你一起长大**。语音笔记的重点不是今天这条录音,而是三年后你的库长什么样——去年三月遛狗时冒出来的念头,帮你今天写出一篇东西;一年前的半个想法,后来成为下个月项目的种子。所以我们让产品保持小,让团队保持精简,让现金跑得久。我们自己每天都在用,这是「需要你的时候它会在」唯一诚实的理由。

如果其中任何一项是核心,继续用 Voicenotes。
