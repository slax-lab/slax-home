## AudioPen 做得好的地方

直说。作为一款单人产品(Louis Pereira 从 2023 年开始独立运营)，AudioPen 迭代很快，覆盖很广。

突出之处：

- **重写强度**——三档，行为差异真实可见。低 = 只清理转录文本。中 = 用你选的风格重写，但保留你的词汇和语气。高 = 完全重写，风格和长度任你选
- **Mac 应用带系统级语音输入**——M1+ Mac 上是真正的生产力解锁。在系统任何文本框里讲话，得到干净文本。Slax Note 没有这个
- **Chrome 扩展**——又一个采集入口
- **音频文件上传**——丢一个 100MB 录音进去，走同一条重写流水线。Prime 允许每 4 周 100 个文件
- **SuperSummaries**——把多条笔记合一。每周回顾时有用
- **重写已有笔记 + 处理粘贴文本**——AI 流水线不绑定「新录音」
- **Zapier + Webhook**——把成品笔记推到 Notion、Obsidian、邮件，哪儿都行
- **一次性定价**——付一次，拥有那个窗口。续期前没有月费

如果你每天的工作流是「语音 → 抛光段落 → 推到别处」，AudioPen 站得住脚。

## Slax Note 走的不同

短版：AudioPen 重写**你的**话，Slax Note 把它塑形成你真要发的那些渠道——而且做这个塑形的 Prompt 是可分享的。

### 能在用户之间流动的 Prompt

AudioPen 的写作风格覆盖正式邮件、法律体、要点等等——加上用户自建的自定义风格。自定义风格留在你自己账号里。

Slax Note 的 Prompt 是可分享的。你自己写一条——给你想要的输出的完整指令——可以分享给别的 Slax Note 用户：

- *「把这个变成 280 字符以内的推文，开头要锋利。」*
- *「做成一条 LinkedIn 帖子，200 字，职业语气。」*
- *「重写成博客初稿，带小标题。」*
- *「把里面的决策项单独列成 bullet list。」*

同一段音频，四种输出，并排呈现。你保留想要的。

### Restyle 从入门档就有

AudioPen 的「Restyle 已有笔记」——给已有录音应用另一种重写——是**仅 Prime**的功能。Slax Note 的 Prompt 驱动重写从入门档就有。如果你常活在「我想把这段录音再换个形状」这个循环里，这条很重要。

### 定价姿态

AudioPen 是一次性、定窗口：33 美元 / 3 个月，99 美元 / 1 年，159 美元 / 2 年。Slax Note 是月度订阅，随时取消：4.99 美元/月或 49.99 美元/年。

年成本大致相当。承诺形态不同。

如果其中任何一项是你日常工作流的核心，继续用 AudioPen——或者两个一起用。

## FAQ

**Slax Note 真的能免费试用吗？**  
是的。免费版：5 条保存笔记，共 10 次 AI 润色，录音上限 5 分钟。Pro(4.99 美元/月或 49.99 美元/年)把这些放宽到无限笔记、每天 300 次润色、30 分钟录音。

**一定要放弃 AudioPen 吗？**  
不必。两个并行用几周。AudioPen 做「一段干净文字」和 Mac 语音输入；Slax Note 做「这段录音要变成几种输出」。看你更常打开哪个。

**Slax Note 有 Mac 应用吗？**  
没有。今天有 iOS、Android 和 Web([n.slax.com](https://n.slax.com))。如果 macOS 上的系统级语音输入是核心，AudioPen 的 Mac 应用就是你要的。

**Slax Note 支持什么语言？**  
应用界面有**英文、中文、Português**。语音转录支持英文、中文、德文、日文(以营销页为准)。
