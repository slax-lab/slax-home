## AudioPen 做得好的地方

我們公道一點。作為一個單人營運的產品(由 Louis Pereira 自 2023 年起打造和經營)，AudioPen 出貨速度快、涵蓋範圍廣。

值得提的：

- **重寫強度**——三個層級，行為真的有差別。低 = 只清理逐字稿。中 = 用你選的風格重寫但保留你的詞彙和語氣。高 = 完整重寫，你選風格和長度
- **Mac App 加系統範圍語音輸入**——M1+ Mac 獲得真正的生產力解鎖。在 OS 上任何文字欄位說話，獲得乾淨文字。Slax Note 沒有這個
- **Chrome 擴充功能**——多一個捕捉發生的地方
- **音訊檔案上傳**——丟入 100MB 錄音，獲得相同的重寫流程。Prime 允許每 4 週 100 個檔案
- **SuperSummaries**——把多份筆記合併成一份。每週回顧很實用
- **重新風格化既有筆記 + 處理貼上的文字**——AI 流程不綁定「新錄音」
- **Zapier + webhook**——把完成的筆記接到 Notion、Obsidian、電子信箱、任何地方
- **一次性定價**——付一次，擁有那段時間。續訂前無循環費用

如果你的日常工作流是「memo → 精煉段落 → 推到別處」，AudioPen 名副其實。

## Slax Note 走得不一樣的地方

簡單說：AudioPen 重寫*你的*話；Slax Note 把它們塑形成你實際發布的管道——而且做這個塑形的 Prompt 可以分享。

### 在使用者之間流通的 Prompt

AudioPen 的寫作風格涵蓋正式信件、法律、條列等等——再加上使用者自訂風格。自訂風格留在你自己的帳號裡。

Slax Note 的 Prompt 可分享。你寫自己的——完整描述你想要的輸出——並可以分享給其他 Slax Note 使用者：

- *「把這個變成 280 字推文，開頭犀利。」*
- *「做成 LinkedIn 貼文，200 字，專業語氣。」*
- *「重寫成附小標的部落格草稿。」*
- *「把決定點抽出來變成獨立的條列。」*

同一段音訊，四種輸出，並排呈現。你留下你想要的。

### 重新風格化從入門方案就有

AudioPen 的「重新風格化既有筆記」——對你已經錄的內容套用不同的重寫——是 **Prime 限定**功能。Slax Note 的 Prompt 驅動重寫從入門方案就能用。如果你常常處於「我想對同一段錄音試另一種樣貌」的迴圈裡，這很重要。

### 定價姿態

AudioPen 是一次性、有時段的：$33 / 3 個月、$99 / 1 年、$159 / 2 年。Slax Note 是月訂閱、隨時取消：$4.99/月或 $49.99/年。

大致同樣的年費。不同的承諾形狀。

我們寧可你睜開雙眼切換。AudioPen 有而我們沒有的：

## FAQ

**Slax Note 真的免費可試用嗎？**
是。免費方案：5 則儲存筆記、總共 10 次 AI Polish、錄音長度上限 5 分鐘。Pro($4.99/月或 $49.99/年)把上限提升到無限筆記、每天 300 次潤飾、30 分鐘錄音。

**我必須放棄 AudioPen 嗎？**
不必。並用幾週。AudioPen 做「一個乾淨段落」和 Mac 語音輸入；Slax Note 做「這段錄音需要變成多種輸出」。看你比較常拿哪一個。

**Slax Note 有 Mac App 嗎？**
沒有。今天是 iOS、Android 和 Web([n.slax.com](https://n.slax.com))。如果 macOS 上的系統範圍語音輸入是核心，AudioPen 的 Mac App 是你要的。

**Slax Note 支援哪些語言？**
App UI 是 **English、中文 和 Português**。語音轉錄支援英文、中文、德文和日文(依行銷頁)。
