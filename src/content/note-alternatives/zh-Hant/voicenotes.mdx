## Voicenotes 做得好的地方

我們公道一點。Voicenotes 產品(現在實際上是一個品牌下的三個產品：Meetings、Dictation、Memos)在規模上令人印象深刻。

值得提的：

- **跨完整歷史的 Ask AI。**Pro 的「你的對話永遠可搜尋」是真實的第二大腦承諾
- **廣泛的平台涵蓋。**macOS、Windows、iOS、Android、watchOS、Wear OS、Chrome 擴充功能、Web。在這個類別中比多數競品涵蓋更多平台
- **混合語言自動偵測**——句中切換，無需設定
- **轉錄涵蓋範圍**——網站上 60+ 種語言，商店列表更多
- **「Create / AI Creations」**——8 個內建預設(摘要、待辦事項、信件、訊息、Meeting Report 等)加上自訂 prompt 輸入。儲存的 Prompt 有 Custom 和 Saved 分頁
- **Pro 整合**——Zapier、Obsidian、Notion、Claude、ChatGPT 開箱即用
- **會議錄音加即時報告**——如果你真的做會議筆記，那是頭條功能
- **Dictation 清理**——自動移除填充詞和重複起頭

如果你的工作涉及會議，而你想要一個工具在一個訂閱下處理會議 + 個人筆記 + dictation,Voicenotes 是強選擇。

## Slax Note 走得不一樣的地方

Voicenotes 擴展涵蓋了 meetings + dictation + memos + 整合。Slax Note 堅守在個人語音筆記——並下了一個 Voicenotes 沒下的特定賭注。

### 可以寄給別人的 Prompt

兩個產品都允許你寫自訂 Prompt。Voicenotes 的儲存 Prompt 住在你自己的帳號裡——你可以編輯、重新命名、刪除它們，但它們不離開你的帳號。

Slax Note 的 Prompt 是**可分享的**。你寫一個——*「把這個變成公眾號開頭段落，用我的語氣」*——可以寄給朋友、同事或其他公眾號寫手。他們也能用。

如果你在一個小群人之間寫東西，而那些人一直在說*「可以把你的 prompt 寄給我嗎？」*，我們建構的就是那個工作流。

### 我們不做會議

Voicenotes 的首頁現在主打 Meetings。Memos 是三個子產品之一。

Slax Note 不做會議。我們不做講者分離、行事曆整合或與會者偵測。如果會議是你真正的工作，Voicenotes(或 Otter、Granola、Fireflies、tl;dv)是更好的工具——我們寧可這樣告訴你，也不假裝在競爭。

我們*做的*是個人語音筆記——走路時的想法、日誌式的反思、你想發布的草稿——由你(或你朋友)寫的 Prompt 塑形。

### 價格

Voicenotes 的個人方案(依 iOS IAP)是**月 $14.99** 或**年 $89.99**($99.99 標準，$89.99 促銷)。他們也有**團隊方案 $9/成員/月**(依官方支援中心)。終身「Believer」方案已停售。

Slax Note Pro 是**月 $4.99** 或**年 $49.99**。

不同的產品，所以價格本身不是對的框架。但如果你不需要會議錄音或 Pro 級整合，你在為很多你不會用的功能付費。

> **「Voicenotes 是包含會議的三產品套件。Slax Note 是專注的個人語音筆記 App，加上可分享的 Prompt。」**

## 我們怎麼想這件事

兩個信念塑造了 Slax Note。兩者都和 Voicenotes 2026 年走的方向形成直接對比。

**先做個人的，不是會議的。**會議工具會把產品拉向講者分離、行事曆整合、與會者偵測、企業定價。每一個都把焦點從那個邊走邊有想法的人身上拉走。Voicenotes 把這些都加上，並把它們做成大門口。我們刻意沒這樣做。Slax Note 是給人的，不是會議室。

**為了五年後還在而做，陪你一起成長。**語音筆記的重點不是今天的錄音。是三年錄音下來你的書庫長什麼樣——去年三月遛狗想到的點子幫你今天寫出某東西、一年前的半個想法變成下個月專案的種子。所以我們把產品做小、團隊做精簡、跑道做長。我們自己每天都在用，這是相信它在你需要時還會在的唯一誠實理由。

## FAQ

**Slax Note 真的免費可試用嗎？**

**Slax Note 像 Voicenotes 一樣做會議錄音嗎？**
不做。我們不做會議。如果你需要會議錄音加報告，Voicenotes(或 Otter / Granola / Fireflies)是對的工具。

**Slax Note 像 Voicenotes 的 Ask AI 一樣可以用 AI 搜尋我的完整歷史嗎？**
今天是基本搜尋。如果跨完整歸檔的 AI 問答是你的主要工作，Voicenotes 目前更強。

**我可以同時用 Voicenotes 和 Slax Note 嗎？**
可以。Voicenotes 做會議和跨完整歸檔搜尋；Slax Note 做個人語音筆記和你和朋友分享的 Prompt。幾週後再決定留誰。

**Slax Note 支援哪些語言？**
App UI 是 **English、中文 和 Português**。語音轉錄支援英文、中文、德文和日文，依行銷頁。
