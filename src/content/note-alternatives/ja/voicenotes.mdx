## Voicenotes の優れている点

正直に言いましょう。Voicenotes のプロダクト（今は実際にひとつのブランド下に 3 つのプロダクト：Meetings、Dictation、Memos）はスコープにおいて印象的です。

際立つもの：

- **全履歴横断の Ask AI。** Pro の「会話は永遠に検索可能」は本物のセカンドブレインの約束
- **広いプラットフォームカバレッジ。** macOS、Windows、iOS、Android、watchOS、Wear OS、Chrome 拡張、Web。このカテゴリーのほとんどの競合より多いプラットフォーム
- **混合言語自動検出** ——文の途中で切り替え、セットアップなし
- **文字起こしのスコープ** ——サイトに 60+ 言語、ストアリスティングにそれ以上
- **「Create／AI Creations」** ——8 つの組み込みプリセット（要約、To-Do リスト、メール、メッセージ、Meeting Report 等）プラスカスタムプロンプト入力。Saved Prompts は Custom と Saved タブを取得
- **Pro 統合**——Zapier、Obsidian、Notion、Claude、ChatGPT を箱から
- **会議録音 ＋ 即時レポート**——実際にミーティングノートを取るなら、それがヘッドライン機能
- **Dictation クリーンアップ**——フィラーワードと言い直しの自動削除

仕事が会議に関わり、会議 ＋ 個人ノート ＋ ディクテーションをひとつのサブスクで扱うひとつのツールが欲しいなら、Voicenotes は強い選択。

## Slax Note の違い

Voicenotes は会議 ＋ ディクテーション ＋ Memos ＋ 統合をカバーするように広がりました。Slax Note は個人ボイスノートに集中したまま——そして Voicenotes がしなかったひとつの特定の賭けをしました。

### 他の人に送れるプロンプト

両方のプロダクトでカスタムプロンプトを書けます。Voicenotes の Saved Prompts はアカウント内に留まります——編集、名前変更、削除はできますが、アカウントを離れません。

Slax Note のプロンプトは**共有可能**です。自分で書く——*「これを公众号の冒頭段落に、私の声で。」* ——友達、チームメイト、または別の公众号の書き手に送れます。彼らも使えます。

*「あなたのプロンプトを送って」*と言い合う小さな人々のグループの中で書いているなら、それが私たちが作ったワークフローです。

### 私たちは会議をやらない

Voicenotes のホームページは今 Meetings をメインに据えています。Memos は 3 つのサブプロダクトのひとつ。

Slax Note は会議をやりません。話者分離、カレンダー統合、出席者検出はやりません。会議が実際の仕事なら、Voicenotes（あるいは Otter、Granola、Fireflies、tl;dv）の方が良いツール——競合するふりをするより、そう伝えたい。

私たちが*やる*のは個人ボイスノート——歩きながらのアイデア、ジャーナル風の振り返り、出版したいドラフト——あなた（または友達）が書いたプロンプトで形作る。

### 価格

Voicenotes の個人プラン（iOS IAP）は **$14.99/月**または **$89.99/年**（$99.99 標準、$89.99 プロモ）。**$9/メンバー/月のチームプラン**もあります（公式ヘルプセンターに基づく）。生涯「Believer」ティアは廃止。

Slax Note Pro は **$4.99/月**または **$49.99/年**。

違うプロダクトなので価格だけが正しいフレームではありません。でも、会議録音や Pro ティア統合が要らないなら、使わないたくさんの能力に払っていることになります。

> **「Voicenotes は会議を含む 3 プロダクトのスイート。Slax Note は共有可能プロンプト付きの集中した個人ボイスノートアプリ。」**

## 私たちの考え方

Slax Note を形作る 2 つの信念。両方とも、Voicenotes が 2026 年に向かった方向とは直接の対比です。

**個人優先、会議優先ではない。** ミーティングツールはプロダクトを話者分離、カレンダー統合、出席者検出、エンタープライズ価格の方向に引っ張ります。それぞれが、アイデアを抱えて道を歩いている人から焦点を引き離します。Voicenotes はそれをすべて追加し、玄関にしました。私たちは意図的にしなかった。Slax Note は個人のため、会議室のためではない。

**5 年後もここにいる、あなたと一緒に育つように作る。** ボイスノートの要点は今日の録音ではありません。3 年間の録音の後にライブラリがどう見えるか——3 月の犬の散歩の思いつきが今日の何かを書く助けになる、1 年前の半端な考えが来月のプロジェクトの種であることが分かる。だからプロダクトを小さく、チームをリーンに、ランウェイを長く保ちます。自分たちで毎日使っているので、必要なときにそこにあると信じる唯一の誠実な理由です。

## Slax Note にまだないもの

両眼を開けて切り替えてほしい。Voicenotes が今日できて私たちができないこと：

- **会議録音 ＋ レポート**——完全に私たちのスコープ外。会議はやらない
- **フル履歴横断の Ask AI** ——私たちの検索は基本的。「今まで録音したすべてを検索」がメインの仕事なら、現在 Voicenotes の方が良い
- **macOS、Windows、Apple Watch、Wear OS、Chrome 拡張**——私たちは iOS、Android、Web を持っています。それだけ
- **Zapier／Obsidian／Notion／Claude／ChatGPT 統合**——未構築。共有は share-to でアプリ内
- **長い録音** ——録音ごとに 30 分上限。Voicenotes の Free Basic だけで 1 時間の会議を許可
- **フィラーワードクリーンアップ付きの別プロダクトとしての Dictation**

これらが中心なら、Voicenotes を続けて。

## よくある質問

**Slax Note は本当に無料で試せる？**
はい。無料プラン：5 件のノート保存、合計 10 回の AI Polish、5 分までの録音。Pro（$4.99/月または $49.99/年）でその制限が外れます。

いいえ。会議はやりません。レポート付きの会議録音が必要なら、Voicenotes（あるいは Otter／Granola／Fireflies）が正しいツール。

**Slax Note は Voicenotes の Ask AI のように AI で全履歴を検索できる？**
今日は基本検索。フルアーカイブ AI Q&A がメインの仕事なら、現在 Voicenotes の方が強い。

**Voicenotes と Slax Note を一緒に使える？**
はい。Voicenotes を会議とフルアーカイブ検索に、Slax Note を個人ボイスノートと友達と共有するプロンプトに。数週間後に残すものを決めて。

**Slax Note は何の言語をサポート？**
アプリ UI は**英語、中文、Português**。音声文字起こしは英語、中国語、ドイツ語、日本語をマーケティングページに基づきサポート。
