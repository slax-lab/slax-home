## AudioPen の優れている点

正直に言いましょう。一人運営のプロダクト（Louis Pereira が 2023 年から構築・運営）として、AudioPen は速く出荷し、多くをカバーしています。

際立つもの：

- **Rewriting Intensity** ——3 レベルで本物の、明確に区別された挙動。Low ＝整えた文字起こしのみ。Medium ＝選んだスタイルで書き直しつつ語彙と口調を保つ。High ＝完全書き直し、スタイルと長さを選択
- **システム全体のボイスタイピング付き Mac アプリ**——M1+ Mac で本物の生産性アンロック。OS のどのテキストフィールドでも話してきれいなテキストを得られる。Slax Note にはこれがない
- **Chrome 拡張**——キャプチャの場所がもうひとつ
- **音声ファイルアップロード**——100MB の録音を投入し、同じ書き直しパイプラインを通せる。Prime は 100 ファイル／4 週間
- **SuperSummaries** ——複数ノートをひとつに結合。週次レビューに便利
- **既存ノートを Restyle ＋貼り付けテキストの処理**——AI パイプラインは「新規録音」に縛られていない
- **Zapier ＋ Webhook** ——完成したノートを Notion、Obsidian、メール、どこへでもパイプ
- **買い切り価格**——一度払い、ウィンドウを所有。更新まで継続課金なし

日々のワークフローが「メモ → きれいな段落 → 他へ送る」なら、AudioPen はその場所を勝ち取っています。

## Slax Note の違い

短く：AudioPen は*あなた*の発話を書き直す。Slax Note はそれを、あなたが実際に投稿するチャンネルに形作る——そして形作るプロンプトは共有可能。

### ユーザー間を旅するプロンプト

AudioPen の Writing Styles はフォーマルなメール、法務、箇条書きなどをカバー——プラスユーザー作成のカスタムスタイル。カスタムスタイルは自分のアカウント内に留まります。

Slax Note のプロンプトは共有可能です。自分で書く——欲しい出力のためのフル指示——他の Slax Note ユーザーと共有できます：

- *「これを 280 文字のツイートに、シャープな冒頭で。」*
- *「これを 200 語の LinkedIn 投稿に、プロフェッショナルな口調で。」*
- *「サブヘッド付きのブログドラフトに書き直して。」*
- *「決定事項を別の箇条書きにまとめて。」*

同じ音声、4 つの出力、並列で。欲しいものをキープ。

### Restyle はエントリーティアから

AudioPen の「既存ノートを Restyle」——すでに作った録音に別の書き直しを適用する——は **Prime のみ**の機能です。Slax Note のプロンプト駆動書き直しはエントリーティアから動きます。「同じ録音を別の形でも試したい」ループに住んでいるなら、これは効きます。

### 価格の姿勢

AudioPen は買い切りで期間制限：$33／3 か月、$99／1 年、$159／2 年。Slax Note は月額サブスクでいつでもキャンセル可能：$4.99/月または $49.99/年。

年単位ではほぼ同じコスト。コミットメントの形が違います。

## 価格——形が違う

**AudioPen** は買い切り、期間制限プランを売っています：**$33／3 か月**、**$99／1 年**、**$159／2 年**。月額なし。期限が来たら更新。
**Slax Note Pro**：**$4.99/月**または **$49.99/年**。いつでもキャンセル可能。

コミットメント形の価格と既知の単発コストが欲しいなら、AudioPen のモデルが親切。月単位で試して合わなければやめたいなら、Slax Note のモデルが親切。

> **「AudioPen は洗練されたペンを売る。Slax Note は出荷先のチャンネルを売る。」**

## 私たちの考え方

Slax Note を形作る 2 つの信念。

**個人優先、会議優先ではない。** ミーティングツールはプロダクトを話者分離、カレンダー統合、出席者検出、エンタープライズ価格の方向に引っ張ります——それぞれが、アイデアを抱えて道を歩いている人から焦点を引き離します。AudioPen と Slax Note はどちらも個人優先の賭けをしました、ただ違うチームから。

**5 年後もここにいる、あなたと一緒に育つように作る。** ボイスノートの要点は今日の録音ではありません。3 年後にライブラリがどう見えるか——3 月の犬の散歩の思いつきが今日の何かを書く助けになる、1 年前の半端な考えが来月のプロジェクトの種であることが分かる。だからプロダクトを小さく、チームをリーンに、ランウェイを長く保ちます。自分たちで毎日使っているので、必要なときにそこにあると信じる唯一の誠実な理由です。

## Slax Note にまだないもの

両眼を開けて切り替えてほしい。AudioPen にあって私たちにないもの、今日：

- **システム全体のボイスタイピング付き Mac アプリ**——ありません。macOS の多くのアプリで書き、すべてに口述したいなら、AudioPen の Mac アプリは本物の差別化
- **Chrome 拡張**——未構築
- **Zapier ＋ Webhook** ——未構築。共有は今日アプリ内で起きる
- **SuperSummaries** ——複数ノートをひとつに結合はまだ未搭載
- **既存ノートを Restyle ／貼り付けテキストの処理**——Slax Note のパイプラインは今日、新規音声入力中心
- **長い録音**——Slax Note は録音ごとに 30 分上限。AudioPen Prime は録音ごとに 15 分上限ですが、100MB までの音声ファイルアップロードを許可するので、本物の長尺はそちらが扱いやすい

これらが日々のワークフローの中心なら、AudioPen を続けるか、両方使ってください。

## よくある質問

**Slax Note は本当に無料で試せる？**
はい。無料プラン：5 件のノート保存、合計 10 回の AI Polish、5 分までの録音。Pro（$4.99/月または $49.99/年）でその制限が外れ、無制限のノート、1 日 300 回の polish、30 分の録音になります。

**AudioPen を捨てなければならない？**
いいえ。数週間両方を動かしてください。AudioPen を「きれいな一段落」と Mac ボイスタイピングに、Slax Note を「この録音は複数の出力になる必要がある」用に。どちらにより手が伸びるかで決めて。

**Slax Note に Mac アプリはある？**
ありません。iOS、Android、Web（[n.slax.com](https://n.slax.com)）が今日。macOS のシステム全体のボイスタイピングが中心なら、AudioPen の Mac アプリが欲しいものです。

**Slax Note は何の言語をサポート？**
アプリ UI は**英語、中文、Português**。音声文字起こしは英語、中国語、ドイツ語、日本語をサポート（マーケティングページに基づく）。
