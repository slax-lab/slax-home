## AudioPen 이 잘하는 것

솔직히 말씀드리면, 1 인 운영 제품 (2023 년부터 Louis Pereira 가 만들고 운영) 으로서 AudioPen 은 빠르게 출시하고 많은 영역을 커버합니다.

눈에 띄는 것:

- **재작성 강도** — 세 레벨, 실제로 다른 동작. Low = 정리된 녹취록만. Medium = 선택한 스타일로 재작성하되 어휘와 톤 유지. High = 완전 재작성, 스타일과 길이 본인 선택
- **시스템 전체 음성 타이핑이 있는 Mac 앱** — M1+ Mac 에 진짜 생산성 잠금 해제. OS 의 어떤 텍스트 필드에서도 말하면 깔끔한 텍스트. Slax Note 에는 이게 없음
- **Chrome 확장** — 캡처 가능한 곳이 하나 더
- **오디오 파일 업로드** — 100MB 녹음을 드롭하면 같은 재작성 파이프라인. Prime 은 4 주에 100 파일 허용
- **SuperSummaries** — 여러 노트를 하나로 병합. 주간 리뷰에 유용
- **기존 노트 Restyle + 붙여넣은 텍스트 처리** — AI 파이프라인이 "새 녹음" 에 묶이지 않음
- **Zapier + 웹훅** — 완성된 노트를 Notion, Obsidian, 이메일, 어디로든 파이프
- **일회성 가격** — 한 번 결제, 그 기간 소유. 갱신 전까지 반복 청구 없음

일상 워크플로가 "메모 → 다듬어진 단락 → 다른 곳으로 보내기" 라면 AudioPen 은 마땅한 자리를 얻었습니다.

## Slax Note 가 어떻게 다른가

짧게: AudioPen 은 *본인의* 말을 재작성합니다. Slax Note 는 그것을 실제로 게시할 채널들의 모양으로 빚어 줍니다 — 그리고 그 모양을 만드는 프롬프트가 공유 가능합니다.

### 사용자 간에 이동하는 프롬프트

AudioPen 의 글쓰기 스타일은 공식 이메일, 법률, 글머리 기호 같은 것들을 커버합니다 — 사용자 생성 커스텀 스타일도 포함. 커스텀 스타일은 본인 계정 안에 머뭅니다.

Slax Note 의 프롬프트는 공유 가능합니다. 원하는 출력에 대한 전체 지침을 직접 쓰고, 다른 Slax Note 사용자와 공유할 수 있습니다:

- *"이걸 280 자 트윗으로 만들어 줘, 강한 시작 문구."*
- *"이걸 LinkedIn 게시물로 만들어 줘, 200 단어, 전문적 톤."*
- *"부제목이 있는 블로그 초안으로 다시 써 줘."*
- *"결정 사항을 별도 글머리 기호 목록으로 뽑아 줘."*

같은 오디오, 네 개의 출력, 나란히. 원하는 것을 보관합니다.

### Restyle 이 입문 티어부터 포함

AudioPen 의 "기존 노트 Restyle" — 이미 만든 녹음에 다른 재작성을 적용 — 은 **Prime 전용** 기능입니다. Slax Note 의 프롬프트 기반 재작성은 입문 티어부터 동작합니다. "같은 녹음을 다른 모양으로 시도해 보고 싶다" 의 루프에 산다면 중요합니다.

### 가격 자세

AudioPen 은 일회성이고 시간 박스: 3 개월 33 달러, 1 년 99 달러, 2 년 159 달러. Slax Note 는 월간 구독이며 언제든 취소 가능: 월 4.99 달러 또는 연 49.99 달러.

대략 같은 연간 비용. 다른 약정 모양.

## 저희가 생각하는 방식

두 가지 믿음이 Slax Note 를 만들었습니다.

**미팅이 아니라 개인 우선.** 미팅 도구는 제품을 화자 분리, 캘린더 통합, 참석자 감지, 엔터프라이즈 가격으로 끌고 갑니다 — 각각이 길에서 아이디어와 함께 걷는 사람으로부터 멀어지게 합니다. AudioPen 과 Slax Note 둘 다 개인 우선의 베팅을 했습니다. 단지 다른 팀에서.

**5 년 후에도 여기 있도록, 여러분과 함께 자라도록 만들어졌습니다.** 음성 노트의 요점은 오늘의 녹음이 아닙니다. 3 년 후의 라이브러리의 모습입니다 — 지난 3 월 산책길에서의 아이디어가 오늘 무언가 쓰게 돕고, 1 년 전 반쪽 생각이 다음 달 프로젝트의 씨앗이 되는 것. 그래서 제품은 작게, 팀은 린(lean)하게, 런웨이는 길게 유지합니다. 저희도 매일 직접 사용하며, 그것이 필요할 때 여전히 거기 있을 거라고 믿을 유일한 솔직한 이유입니다.

## Slax Note 가 아직 가지지 않은 것

두 눈을 다 뜨고 갈아타시는 게 낫습니다. AudioPen 에 있고 저희에게 없는 것들, 오늘:

- **시스템 전체 음성 타이핑이 있는 Mac 앱** — 없음. macOS 에서 여러 앱에 걸쳐 글을 쓰고 그 모두에 받아쓰기하고 싶다면 AudioPen 의 Mac 앱은 진짜 차별화 요소
- **Chrome 확장** — 미구현
- **Zapier + 웹훅** — 미구현. 공유는 오늘 앱 안에서 일어남
- **SuperSummaries** — 여러 노트를 하나로 결합하는 기능은 아직 제품에 없음
- **기존 노트 Restyle / 붙여넣은 텍스트 처리** — Slax Note 의 파이프라인은 오늘 새 음성 입력 중심
- **긴 녹음** — Slax Note 는 녹음당 30 분으로 제한. AudioPen Prime 은 녹음당 15 분 제한이지만 100MB 까지 오디오 파일 업로드 허용, 그래서 진짜 긴 형식은 거기가 더 접근 가능

이 중 어떤 것이 일상 워크플로의 핵심이라면 AudioPen 과 함께 머무세요 — 또는 둘 다 운영하세요.

**Slax Note 는 정말 무료로 사용해 볼 수 있나요?**
예. 무료 플랜: 5 개 저장 노트, 총 10 AI Polish 실행, 최대 5 분 녹음. Pro (월 4.99 달러 또는 연 49.99 달러) 가 무제한 노트, 일 300 polish, 30 분 녹음으로 한도를 올립니다.

**AudioPen 을 포기해야 하나요?**
아닙니다. 몇 주간 둘 다 운영하세요. "깔끔한 단락 하나" 와 Mac 음성 타이핑에는 AudioPen, "이 녹음이 여러 출력이 되어야 함" 에는 Slax Note. 어느 쪽으로 더 자주 손이 가는지 보세요.

**Slax Note 는 Mac 앱이 있나요?**
아닙니다. iOS, Android, 그리고 웹 ([n.slax.com](https://n.slax.com)) 만 오늘. macOS 에서 시스템 전체 음성 타이핑이 핵심이라면 AudioPen 의 Mac 앱이 원하는 것입니다.

**Slax Note 는 어떤 언어를 지원하나요?**
앱 UI 는 **영어, 中文, Português** 입니다. 음성 받아쓰기는 영어, 중국어, 독일어, 일본어를 지원합니다 (마케팅 페이지 기준).
