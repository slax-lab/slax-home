Voicenotes 는 한때 "먼저 말하고 나중에 정리" 의 단순한 음성 노트 앱이었습니다. 2026 년 홈페이지에는 이제 **"Voicenotes Meetings — 녹음, 받아쓰기, 즉시 미팅 보고서 받기"** 라고 적혀 있습니다. 원래의 개인 노트 제품은 여전히 **Memos** 서브 제품으로 존재합니다 (Meetings 와 Dictation 옆에) — 그러나 더 이상 정문이 아닙니다.

개인 음성 노트를 위해 Voicenotes 에 왔고 실제로 미팅 노트를 하지 않는다면, 그 리브랜드가 아마 무언가 말해 줄 겁니다.

Slax Note 는 다른 베팅을 했습니다. **개인 음성 노트** 에 집중을 유지했고, **커스텀 프롬프트를 공유 가능** 하게 만들었습니다 — 본인 것을 쓰고 친구에게 (또는 팀원에게, 또는 다른 公众号 작가에게) 보내 그들도 사용할 수 있게. **하나의 녹음, 여러 모양 — 본인과 친구가 쓴 프롬프트로 빚어짐.**

## Voicenotes 가 잘하는 것

솔직히 말씀드리면, Voicenotes 제품 (이제 한 브랜드 아래 실제로 세 제품: Meetings, Dictation, Memos) 은 범위가 인상적입니다.

눈에 띄는 것:

- **전체 히스토리에 걸친 Ask AI.** Pro 의 "대화가 영원히 검색 가능하게 머뭅니다" 는 진짜 세컨드 브레인 약속
- **넓은 플랫폼 커버리지.** macOS, Windows, iOS, Android, watchOS, Wear OS, Chrome 확장, 웹. 이 카테고리의 대부분 경쟁자보다 더 많은 플랫폼
- **혼합 언어 자동 감지** — 설정 없이 문장 중간에 전환
- **받아쓰기 범위** — 사이트에 60+ 언어, 스토어 목록에 더 많음
- **"Create / AI Creations"** — 8 개 빌트인 프리셋 (요약, 할 일 목록, 이메일, 메시지, 미팅 보고서 등) 더하기 커스텀 프롬프트 입력. 저장된 프롬프트는 Custom 과 Saved 탭을 받음
- **Pro 통합** — Zapier, Obsidian, Notion, Claude, ChatGPT 기본 제공
- **즉시 보고서가 있는 미팅 녹음** — 실제로 미팅 노트를 한다면 헤드라인 기능
- **받아쓰기 정리** — 필러 단어와 잘못된 시작 자동 제거

업무가 미팅을 포함하고 미팅 + 개인 노트 + 받아쓰기를 한 구독으로 처리하는 도구를 원한다면 Voicenotes 는 강한 선택입니다.

## Slax Note 가 어떻게 다른가

Voicenotes 는 미팅 + 받아쓰기 + 메모 + 통합을 커버하도록 확장했습니다. Slax Note 는 개인 음성 노트에 집중을 유지했고 — Voicenotes 가 하지 않은 한 가지 구체적인 베팅을 했습니다.

### 다른 사람에게 보낼 수 있는 프롬프트

두 제품 모두 커스텀 프롬프트를 쓰게 합니다. Voicenotes 의 저장된 프롬프트는 본인 계정에 삽니다 — 편집, 이름 변경, 삭제할 수 있지만 계정을 떠나지 않습니다.

Slax Note 의 프롬프트는 **공유 가능합니다**. 하나를 쓰고 — *"내 목소리로 公众号 시작 단락으로 만들어 줘"* — 친구, 팀원, 또는 다른 公众号 작가에게 보낼 수 있습니다. 그들도 사용할 수 있습니다.

*"네 프롬프트 보내 줄 수 있어?"* 라고 계속 말하는 소규모 사람들 안에서 글을 쓴다면, 그것이 저희가 만든 워크플로입니다.

### 미팅을 하지 않음

Voicenotes 의 홈페이지는 이제 Meetings 로 시작합니다. Memos 는 세 개의 서브 제품 중 하나입니다.

Slax Note 는 미팅을 하지 않습니다. 화자 분리, 캘린더 통합, 참석자 감지를 하지 않습니다. 미팅이 실제 일이라면 Voicenotes (또는 Otter, Granola, Fireflies, tl;dv) 가 더 나은 도구입니다 — 경쟁하는 척하기보다 그렇게 말씀드리는 게 낫습니다.

저희가 *하는* 것은 개인 음성 노트입니다 — 걸으며 떠오르는 아이디어, 일기 스타일 회고, 게시하고 싶은 초안 — 본인 (또는 친구) 이 쓴 프롬프트로 빚어짐.

### 가격

Voicenotes 의 개인 플랜 (iOS IAP 기준) 은 **월 14.99 달러** 또는 **연 89.99 달러** (99.99 달러 표준, 89.99 달러 프로모션). **멤버당 월 9 달러의 팀 플랜** 도 있습니다 (공식 도움말 센터 기준). 평생 "Believer" 티어는 폐지되었습니다.

Slax Note Pro 는 **월 4.99 달러** 또는 **연 49.99 달러** 입니다.

다른 제품이므로 가격만으로는 맞는 프레임이 아닙니다. 그러나 미팅 녹음이나 Pro 티어 통합이 필요하지 않다면 사용하지 않을 많은 역량에 비용을 내는 셈입니다.

> **"Voicenotes 는 미팅을 포함하는 3 제품 스위트입니다. Slax Note 는 공유 가능한 프롬프트가 있는 집중된 개인 음성 노트 앱입니다."**

## 저희가 생각하는 방식

두 가지 믿음이 Slax Note 를 만들었습니다. 둘 다 Voicenotes 가 2026 년에 간 방향과 직접 대비됩니다.

**미팅이 아니라 개인 우선.** 미팅 도구는 제품을 화자 분리, 캘린더 통합, 참석자 감지, 엔터프라이즈 가격으로 끌고 갑니다. 각각이 길에서 아이디어와 함께 걷는 사람으로부터 집중을 멀어지게 합니다. Voicenotes 는 그 모든 것을 추가하고 정문으로 만들었습니다. 저희는 의도적으로 그러지 않았습니다. Slax Note 는 사람을 위한 것이지 미팅 룸을 위한 것이 아닙니다.

**5 년 후에도 여기 있도록, 여러분과 함께 자라도록 만들어졌습니다.** 음성 노트의 요점은 오늘의 녹음이 아닙니다. 3 년의 녹음 후 라이브러리의 모습입니다 — 지난 3 월 산책길에서의 아이디어가 오늘 무언가 쓰게 돕고, 1 년 전 반쪽 생각이 다음 달 프로젝트의 씨앗이 되는 것. 그래서 제품은 작게, 팀은 린(lean)하게, 런웨이는 길게 유지합니다. 저희도 매일 직접 사용하며, 그것이 필요할 때 여전히 거기 있을 거라고 믿을 유일한 솔직한 이유입니다.

## Slax Note 가 아직 가지지 않은 것

두 눈을 다 뜨고 갈아타시는 게 낫습니다. Voicenotes 가 오늘 하는데 저희가 하지 않는 것:

- **미팅 녹음 + 보고서** — 저희 범위 밖. 미팅은 하지 않음
- **전체 히스토리에 걸친 Ask AI** — 저희 검색은 기본. "내가 녹음한 모든 것을 검색" 이 주요 일이라면 Voicenotes 가 현재 더 낫습니다
- **macOS, Windows, Apple Watch, Wear OS, Chrome 확장** — iOS, Android, 그리고 웹 보유. 그게 전부
- **Zapier / Obsidian / Notion / Claude / ChatGPT 통합** — 미구현. 공유는 앱 내 공유 기능을 통해
- **긴 녹음** — 녹음당 30 분 제한. Voicenotes 의 무료 Basic 만 해도 1 시간 미팅 허용
- **별도 제품으로서의 받아쓰기** + 필러 단어 정리

이 중 어떤 것이 핵심이라면 Voicenotes 와 함께 머무세요.

## 자주 묻는 질문

**Slax Note 는 정말 무료로 사용해 볼 수 있나요?**
예. 무료 플랜: 5 개 저장 노트, 총 10 AI Polish 실행, 최대 5 분 녹음. Pro (월 4.99 달러 또는 연 49.99 달러) 가 그 한도를 올립니다.

**Voicenotes 도 커스텀 프롬프트가 있나요?**
예 — 8 개 빌트인 프리셋과 커스텀 프롬프트 입력이 있는 "Create / AI Creations" 라고 부르고, 본인 것을 관리하는 Custom 과 Saved 탭이 있습니다. 차이: Voicenotes 의 저장된 프롬프트는 계정에 머묾; Slax Note 의 것은 다른 사용자와 공유 가능.

**Slax Note 가 Voicenotes 처럼 미팅 녹음을 하나요?**
아닙니다. 미팅을 하지 않습니다. 보고서가 있는 미팅 녹음이 필요하다면 Voicenotes (또는 Otter / Granola / Fireflies) 가 올바른 도구입니다.

**Slax Note 가 Voicenotes 의 Ask AI 처럼 전체 히스토리를 AI 로 검색할 수 있나요?**
오늘은 기본 검색입니다. 전체 아카이브 AI Q&A 가 주요 일이라면 Voicenotes 가 현재 더 강력합니다.

**Voicenotes 와 Slax Note 를 함께 운영할 수 있나요?**
예. 미팅과 전체 아카이브 검색에는 Voicenotes, 개인 음성 노트와 친구와 공유하는 프롬프트에는 Slax Note. 몇 주 후 무엇이 남을지 결정하세요.

**Slax Note 는 어떤 언어를 지원하나요?**
앱 UI 는 **영어, 中文, Português** 입니다. 음성 받아쓰기는 마케팅 페이지 기준으로 영어, 중국어, 독일어, 일본어를 지원합니다.
