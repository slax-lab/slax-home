我們是一個小團隊,正在做 Slax Reader,一款開源的「稍後讀」App,主要做兩件事:

1. 永久保存網頁內容(就算原文消失也沒關係)。
2. 用內建 AI 工具幫你理解所儲存的內容。

來試用或貢獻請看:[slax-lab 在 GitHub 上](https://github.com/slax-lab)

## 它能做什麼

- **儲存內容**:支援網頁、X 推文串和 YouTube 影片。PDF / 電子報支援即將推出。
- **更快學習**:
  - 對看不懂的詞劃重點 → 不必切換分頁就能立刻看到解釋。
  - 從長文自動產生摘要、心智圖或大綱。
- **整理**:自動標籤,可用關鍵字或語意搜尋。
- **訂閱**:追蹤創作者的公開收藏。例如,如果 Elon Musk 使用 Slax Reader 並公開分享他的書籤,你可以訂閱他的收藏,看看他都在讀和看什麼。

## 為什麼我們要做這個

部分原因是很多網際網路上的連結正在消失。根據 Pew Research,2013 到 2023 年間有 25% 的網頁已經不見了。當連結死了,感覺就像失去了一部分記憶。身為一個讀很多東西的人,我希望自己儲存的內容永遠都能讀得到。

第二個原因是現有工具不是只能儲存,就是要在不同 App 之間切換才能學習。我們想要兩者合一。

## 目前狀態

- 可自架([GitHub 倉庫](https://github.com/slax-lab/slax-reader)),但目前安裝過程有點複雜。我們把一鍵部署列為 v2 的優先項目。
- 免費使用(重度 AI 使用有付費選項)。
