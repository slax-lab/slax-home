Pocket 要關閉了,對於儲存完整文章來說真的很糟。我們做了一個免費、開源的工具——[Slax Reader](https://r.slax.com),可以匯入你的 Pocket 書庫,並保留完整內容(不只是連結!)以及標籤。

文章呈現的樣子和原網站幾乎一模一樣,所以你能保留那份乾淨的閱讀體驗。

來看一眼介面長什麼樣:

![Slax Reader 介面預覽](/reader/images/blog/built-an-open-source-pocket-alternative/quick-peek.png)

我自己把 3000+ 篇文章的書庫丟進去測,匯入過程非常順暢。

既然大家都在 Pocket 這件事上同病相憐,我們對想匯入 Pocket 內容或儲存新內容的早期使用者提供無限儲存空間。全部免費。

整個專案都是開源的([Slax Lab 在 GitHub 上](https://github.com/slax-lab)),我們也在做 Docker / Linux 版本和其他自架選項,因為我們知道很多人非常重視能完全掌握自己的資料。

除了單純的稍後讀 App,我們也在內建一些 AI 工具——像是自動生成的文章摘要/分析,或在不離開頁面的情況下問 AI 關於正在讀的內容。這些現在也都可以免費試用。

![AI 聊天功能](/reader/images/blog/built-an-open-source-pocket-alternative/ai-chat.png)

我真心希望你試試看,尤其如果你是 Pocket 難民。

有人找到其他不錯的替代方案嗎?很想聽聽大家都在用什麼。
